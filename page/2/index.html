番茄炖牛腩

发表于 2024-05-12 分类于 好好吃饭

食材

  • 牛腩(最好牛肋腩) 番茄
  • 大葱 姜 蒜 八角 香叶 香菜
  • 番茄酱 盐 糖 味精

方法

  1. 牛腩(1000g)切大块，西红柿(700g)去皮切大丁
  2. 起锅加水，牛腩冷水下锅，焯水。同时准备一高压锅，加入适量水，加入大葱(50g)、姜(30g)、八角(5g)、香叶(3g)后煮开
  3. 撇干净血沫后，将牛腩放入高压锅中，焯水的汤一并倒入高压锅，高压锅上汽煮30分钟，煮好后捞出各种调料
  4. 起锅烧油，下葱花(大葱50g)蒜片(15g)，煸炒至葱花变黄
  5. 倒入西红柿，再加一些番茄酱，简单翻炒
  6. 煮好的牛腩连着煮牛腩的汤一起倒入锅中，大火烧开
  7. 加入盐（6g）、糖(15g)、味精(1g)定味
  8. 关小火，煮至各种食材味道混为一体(大概3~5分钟)，放香菜出锅

注意事项

  • 买牛腩尽可能买有肉有筋的牛腩，口感和香味会更好
  • 牛肉焯水时要冷水下锅，焯水至没有血沫，再转移到高压锅中煮
  • 番茄酱用于增加番茄的香味，只有番茄会感觉番茄味道寡淡
  • 定味时加入糖用于中和番茄的酸味，如果感觉酸了，可以适当增加糖的用量
  • 盐作为底味，用量要准，可以少量多次添加，感觉味道比较飘或者酸味正好没有甜味，可以适量补盐

醋炒蛋

发表于 2024-05-11 分类于 好好吃饭

食材

  • 鸡蛋
  • 剁椒 鲜辣椒 蒜 葱
  • 酱油 醋 糖 味精 香油 淀粉

方法

  1. 调制糖醋汁：50g酱油、60g醋、2g味精、6g糖、15g水(对应8个鸡蛋)
  2. 鸡蛋打入碗中，倒入辣椒(8g剁椒、8g鲜辣椒)
  3. 起锅烧油，开大火，油热倒鸡蛋，将鸡蛋黄戳破，扒拉辣椒使其在蛋液中分布均匀
  4. 鸡蛋底部稍微成壳时，关小火，煎鸡蛋至底部硬壳的溏心状态，盛至容器中
  5. 起锅烧油，油热加入蒜末(15g)、葱花(10g)，炒香后倒入调制好的糖醋汁，熬煮至葱蒜与糖醋汁味道混合
  6. 关小火，勾芡至米粥状态，加入香油(10g)
  7. 将芡汁淋到蛋上，上菜

注意事项

  • 煎鸡蛋时一开始要大火煎出焦壳，然后再转小火煎至溏心
  • 勾芡要少量多次，注意观察

1、分级标题

使用#来对段落等级进行区分，从1~6不等

注意：符号”#”需要与标题之间有一个空格

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
